SMS Spam Classification using TF-IDF
Project Overview

This project builds a machine learning model that classifies SMS messages as Spam or Ham using Natural Language Processing (NLP).

The workflow includes:

**Data cleaning**
Text preprocessing
TF-IDF feature extraction
Model training
Performance evaluation
Dataset
SMS Spam Collection Dataset
**Technologies Used**
Python
Pandas
NumPy
Scikit-learn
NLTK
Matplotlib
**Models Tested**
Logistic Regression
